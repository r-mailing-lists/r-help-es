# r-help-es

R help in Spanish

Automated mbox archive. Updated daily via GitHub Actions.

## Structure

- `raw/` - Monthly .mbox files downloaded from [https://stat.ethz.ch/pipermail/r-help-es/](https://stat.ethz.ch/pipermail/r-help-es/)
- `processed/` - Structured JSON output from rmail-parser
- `meta.json` - List statistics

## Viewer

Browse these archives at [r-mailing-lists.thecoatlessprofessor.com](https://r-mailing-lists.thecoatlessprofessor.com/lists/r-help-es/)
